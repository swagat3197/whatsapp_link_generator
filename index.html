WhatsApp Link Generator

Quickly convert any number into a wa.me link — clean, copy, and share.

Enter the phone number (with or without spaces, dashes, 0 or country code). The tool will sanitize it and create a ready-to-share WhatsApp link.

Format: wa.me/+{country}{number}

Examples accepted: 9967530155, 9967 530 155, 09967530155, 919967530155, +919967530155

Made with ❤️ — glassy UI
How it works
Type or paste the mobile number. The app will:
  1. Remove spaces, dashes, and non-number characters.
  2. Detect country-code or add +91 if a 10-digit Indian number.
  3. Generate a clickable wa.me/+... link.
